Cómo usar el juego

  • Descarga el paquete Jugar.
  • Extrae la carpeta en tu ordenador.
  • Abre una terminal dentro de esa carpeta.
  • Instala las dependencias si es necesario: python -m pip install pgzero
  • Ejecuta: python Juego.py
Importante: el botón Jugar descarga el paquete para tu ordenador, no para reproducirlo dentro del navegador.

Publicar esta página

  • Sube la carpeta del proyecto a un repositorio o hosting estático.
  • Con GitHub Pages, habilita Pages desde la rama main.
  • Selecciona /root como carpeta de publicación.
  • Una vez publicado, el botón Jugar funcionará para los visitantes.

Si quieres, en el siguiente paso puedo dejarte un guion exacto para subirla a GitHub Pages o convertir el proyecto en un archivo .exe para que el botón descargue una app más directa.
